# Personal-expense-tracker
## Description
A terminal-based expense tracker built in Python that allows users to set 
monthly budgets per category, log daily expenses, and generate spending 
reports. Data is saved to local files so nothing is lost between sessions.

## Features
- Set monthly budgets for 6 categories
- Log expenses with auto-captured date
- View all expenses in a formatted table
- Category summary with budget vs spent
- Full report: grand total, largest expense, top category
- Delete any logged expense
- Auto-saves to expenses.txt and budgets.txt

## How to Run
1. Make sure Python 3 is installed
2. Clone this repo:
   git clone https://(git-token)@github.com/YOUR-USERNAME/personal-expense-tracker.git
3. Navigate into the folder:
   cd personal-expense-tracker
4. Run the app:
   python3 tracker.py
